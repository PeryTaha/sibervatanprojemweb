# Siber Vatan Web API & Security Project
Uygulamanın arayüzü, çalışma mantığı ve API entegrasyonlarını içeren detaylı ekran kaydına aşağıdaki bağlantıdan ulaşabilirsiniz:
👉 **[Proje Çalışma Görüntüsü (Loom)](https://www.loom.com/share/1eb6ec05182d4615ad2e6fec31c3ae73)**
Bu depo, ASP.NET Web API mimarisi kullanılarak geliştirilmiş, veri yönetimi ve siber güvenlik konseptlerini birleştiren teknik bir çalışmadır.

## Teknik Mimari (Stack)
* **Back-End:** .NET Framework (ASP.NET Web API 2)
* **Front-End:** AngularJS (Single Page Application - SPA)
* **Data Access:** Entity Framework / ADO.NET
* **UI Framework:** Bootstrap 5.3 & Font-Awesome

## Temel Özellikler
* **RESTful API Design:** Veri alışverişi tamamen JSON formatında asenkron olarak gerçekleştirilmektedir.
* **MVC Pattern:** Proje yapısı Controller, View ve Model katmanları ile organize edilmiştir.
* **Dynamic Routing:** API ve sayfa yönlendirmeleri için özelleştirilmiş RouteConfig yapılandırması mevcuttur.

## Kurulum Talimatları
1. Projeyi yerel dizininize klonlayın.
2. Visual Studio ile `sibervatanprojemweb.sln` dosyasını çalıştırın.
3. `Web.config` içerisindeki SQL Server Connection String bilgisini güncelleyin.
4. NuGet paketlerini geri yükleyin (Restore) ve projeyi derleyin (Build).
